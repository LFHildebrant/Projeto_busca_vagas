# Projeto_busca_vagas

## Requisitos

**JDK 21 instalado**

**Docker Desktop  instalado**

- https://www.docker.com/

**intellij IDE instalado**

- https://www.jetbrains.com/idea/

## Instalação

- **Clone ou baixe o projeto no GitHub**


- https://github.com/LFHildebrant/Projeto_busca_vagas


- **Na pasta do projeto, com o Docker iniciado, dê o seguinte comando**

````
docker-compose up -d
````

- **Rode o container criado**


- **Rode a aplicação pela IDE**

## Cliente 

**Para rodar a plicação do cliente, entre na pasta "busca_vagas_cliente" pela IDE, e clique no ícone do Chrome**
